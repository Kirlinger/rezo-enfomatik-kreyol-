📖 Glosè Rezo Enfòmatik

97 tèm rezo enpòtan — Anglè, Definisyon an Kreyòl, ak Egzanp.

97 tèm
Access Point

Aparèy ki pèmèt koneksyon san fil (Wi-Fi) pou aparèy mobil yo.

Egzanp: Yon AP Ubiquiti konekte 30 laptop nan yon sal klas.
Address Resolution Protocol (ARP)

Pwotokòl ki jwenn adrès MAC ki koresponn ak yon adrès IP nan rezo lokal la.

Egzanp: Lè PC A vle ping PC B, li voye yon ARP Request pou jwenn MAC PC B.
APIPA

Automatic Private IP Addressing — Windows asiye otomatikman 169.254.x.x si DHCP echwe.

Egzanp: Si ou wè 169.254.x.x, DHCP pa travay.
Authentication

Priyorite pou verifye idantite yon itilizatè oswa aparèy anvan ba aksè.

Egzanp: Saisir non itilizatè ak mo de pase se yon otantifikasyon.
Bandwidth

Kantite done ki ka transmèt sou yon koneksyon nan yon segond — enprime an Mbps oswa Gbps.

Egzanp: Koneksyon 100 Mbps ka transfere 100 megabit chak segond.
BGP

Border Gateway Protocol — pwotokòl routing ant sistèm òtònom (ISP yo) sou Entènèt.

Egzanp: BGP kontwole kijan Entènèt rote pakè ant ISP mondyal.
Broadcast

Mesaj voye ak tout aparèy nan yon rezo an menm tan.

Egzanp: ARP Request se yon broadcast — FF:FF:FF:FF:FF:FF.
Broadcast Domain

Gwoup aparèy ki ka resevwa broadcast youn ak lòt.

Egzanp: Yon VLAN defini yon broadcast domain separe.
Cable Management

Òganizasyon kab yo pou fasilite depannaj ak ventilasyon.

Egzanp: Itilize Velcro straps ak patch panel pou bon cable management.
CIDR

Classless Inter-Domain Routing — nòtasyon /xx pou esprime long masque sou-rezo.

Egzanp: 192.168.1.0/24 se nòtasyon CIDR ki montre 24 bit pou rezo.
CIA Triad

Twa pilye sekirite: Confidentialite, Entegrite, Disponibilite.

Egzanp: Sekirize done = asire CIA Triad la konplè.
Client

Aparèy ki mande sèvis bay yon sèvè.

Egzanp: Navigatè entènèt ou se yon kliyan DNS ak HTTP.
Collision Domain

Zòn kote pakè ka rantre an konfli si yo voye an menm tan.

Egzanp: Chak pò switch gen pwòp collision domain li.
Default Gateway

Routeur ki transfere pakè ki prale deyò rezo lokal la.

Egzanp: Si IP Gateway ou 192.168.1.1, pakè Entènèt yo pase la.
DHCP

Dynamic Host Configuration Protocol — ba adrès IP otomatikman ak lòt enfòmasyon rezo.

Egzanp: DHCP Server bay PC ou 192.168.1.50 otomatikman.
DNS

Domain Name System — tradui non tankou google.com an adrès IP.

Egzanp: DNS resoud example.com → 93.184.216.34.
DORA

Pwosesis DHCP: Discover, Offer, Request, Acknowledge.

Egzanp: Kliyan fè Discover, Sèvè ba Offer, Kliyan fè Request, Sèvè ACK.
DoS

Denial of Service — atitid pou rann yon sèvis inaksesib.

Egzanp: Yon atakè inonde yon sèvè ak millyon pakè pou kraze li.
DDoS

Distributed Denial of Service — DoS ki soti nan plizyè sous an menm tan.

Egzanp: Yon botnet 100,000 machin fè DDoS sou yon sit wèb.
DMZ

Demilitarized Zone — zòn rezo ki separe sèvè piblik yo nan rezo entèn.

Egzanp: Sèvè wèb ou nan DMZ — izole nan firewall.
Dynamic Routing

Routing kote routeur yo pataje otomatikman enfòmasyon wout ak protokòl tankou OSPF, RIP.

Egzanp: OSPF otomatikman ajiste wout si yon lyen tonbe.
EIGRP

Enhanced Interior Gateway Routing Protocol — pwotokòl routing Cisco proprietary.

Egzanp: EIGRP pi rapid konvèje pase RIP.
Encapsulation

Pwosesis ki ajoute antèt (headers) bay done nan chak kouch OSI.

Egzanp: Kouch 4 ajoute TCP header, Kouch 3 ajoute IP header.
Ethernet

Teknoloji rezo kablaj ki pi itilize — defini pa IEEE 802.3.

Egzanp: Kab Cat6 Ethernet konekte PC a switch la nan 1 Gbps.
Evil Twin

Yon faux access point ki imiite yon rezo Wi-Fi lejitim pou vòlè done.

Egzanp: Atakè kreye SSID 'Hotel_WiFi_Free' pou trompe vizitè.
Fiber Optic

Kab ki transmèt done ak vitès limyè — trè vit ak distans long.

Egzanp: Koneksyon fibra optik ISP bay 1 Gbps longdistans.
Firewall

Sistèm ki filtre trafik rezo selon règ — pèmèt oswa bloke.

Egzanp: Firewall pfSense bloke tout trafik ki pa soti pò 443 ak 80.
FTP

File Transfer Protocol — pou transfere fichye. Port 21. Pa chifre.

Egzanp: FTP itilize pou mete fichye sou yon sèvè wèb.
Full Duplex

Kominikasyon nan de direksyon an menm tan.

Egzanp: Kab Ethernet modèn travay an full duplex — resevwa ak voye an menm tan.
Gateway

Aparèy ki sèvi kòm pwen antre/sòti pou trafik rezo.

Egzanp: Routeur ou = default gateway pou tout PC lakay.
GNS3

Graphic Network Simulator 3 — simulatè rezo avanse pou pratike CCNA/Network+.

Egzanp: Etidyan itilize GNS3 pou simile rezo Cisco konplè.
Half Duplex

Kominikasyon nan yon direksyon alafwa sèlman.

Egzanp: Walkie-talkie se half duplex — yon moun pale alafwa.
Host

Nenpòt aparèy ki gen yon adrès IP sou yon rezo.

Egzanp: PC, Telefòn, Sèvè — tout se hôte.
HTTP

HyperText Transfer Protocol — pou chaje paj wèb. Port 80. Pa chifre.

Egzanp: Navigatè itilize HTTP pou mande paj wèb.
HTTPS

HTTP Secure — HTTP chifre ak TLS/SSL. Port 443.

Egzanp: https://bank.com itilize HTTPS pou sekirize koneksyon.
Hub

Aparèy Kouch 1 ki voye done a tout pò — obsolète.

Egzanp: Hub voye yon pakè ak 8 pò menm lè 1 pò sèlman ki mande.
ICMP

Internet Control Message Protocol — itilize pou mesaj kontwòl tankou ping.

Egzanp: ping Google = ICMP Echo Request/Reply.
IDS

Intrusion Detection System — detekte atitid sispèk epi leve alèt.

Egzanp: IDS detekte yon scan pò epi avèti administratè.
IPS

Intrusion Prevention System — detekte ak aktif bloke atitid.

Egzanp: IPS otomatikman bloke yon atakè ki tante SQL Injection.
IP

Internet Protocol — pwotokòl ki ba adrès ak rote pakè sou rezo.

Egzanp: IPv4 adrès (32 bit) identifye chak aparèy sou Entènèt.
IPv4

Vèsyon 4 Internet Protocol — 32 bit, notasyon dezimal.

Egzanp: 192.168.1.1 se yon adrès IPv4.
IPv6

Vèsyon 6 Internet Protocol — 128 bit, notasyon heksadesimal.

Egzanp: 2001:0db8::1 se yon adrès IPv6.
ISP

Internet Service Provider — konpayi ki bay aksè Entènèt.

Egzanp: Natcom, Digicel, Turboteks se ISP Ayiti.
LAN

Local Area Network — rezo nan yon ti espas tankou yon bilding.

Egzanp: Rezo yon lekòl oswa yon biwo se yon LAN.
Latency

Retay pou done ale nan sòs rive destinasyon — mezire an ms.

Egzanp: Ping 5ms = latèns twa bon. Ping 500ms = koneksyon lènt.
Lease

Dire tan yon kliyan ka kenbe yon adrès IP DHCP a.

Egzanp: DHCP lease 8 èdtan = PC renye IP chak 8 èdtan.
Loopback

Adrès 127.0.0.1 — itilize pou teste stack TCP/IP lokal.

Egzanp: ping 127.0.0.1 = teste stack TCP/IP pa ou.
MAC Address

Media Access Control — adrès fizik unik sou chak NIC (Kouch 2).

Egzanp: 00:1A:2B:3C:4D:5E se yon adrès MAC.
MAN

Metropolitan Area Network — rezo ki kouvri yon vil.

Egzanp: Rezo fibra yon ISP ki kouvri Pòtoprens se yon MAN.
Modem

Aparèy ki konvèti siyal DSL/kab/fibra ISP a an Ethernet pou routeur ou.

Egzanp: Modem ADSL konekte ak liy telefòn ISP a.
Multicast

Voye done ak yon gwoup espesifik aparèy (pa tout).

Egzanp: OSPF itilize multicast 224.0.0.5 pou kominike ant routeur.
NAT

Network Address Translation — tradui adrès IP prive an piblik.

Egzanp: Routeur lakay ou fè NAT pou tout PC yo pataje yon IP piblik.
NIC

Network Interface Card — kat rezo ki ba yon PC koneksyon fizik.

Egzanp: NIC ou ba ou adrès MAC ou epi konekte ak switch la.
OSPF

Open Shortest Path First — pwotokòl routing dinamik link-state.

Egzanp: Gwo antèpriz itilize OSPF paske li konvèje vit.
PAN

Personal Area Network — rezo nan espas pèsonèl yon moun.

Egzanp: Bluetooth ant telefòn ak ecouteur se yon PAN.
Patch Panel

Pano ak pò yo konekte kab Ethernet pou òganize kadran rezo.

Egzanp: Patch panel nan rack konekte tout pò mural yo nan switch la.
Peer-to-Peer

Achitekti kote aparèy yo egal — chak ka sèvi kòm kliyan ak sèvè.

Egzanp: Torrent sharing se peer-to-peer.
Phishing

Atitid ki itilize imel oswa sit wèb fo pou vòlè done.

Egzanp: Imel fòs 'Bankèt ou: Klike pou renouvle' = phishing.
ping

Kòmand ki voye ICMP pakè pou tès konektivite.

Egzanp: ping 192.168.1.1 = teste si routeur la reponn.
Port

Nimewo (0-65535) ki identifye yon sèvis espesifik sou yon òdinatè.

Egzanp: HTTP = port 80, HTTPS = 443, SSH = 22.
Proxy

Entèmedyè ant kliyan ak sèvè — ka kache, filtre, oswa cache.

Egzanp: Proxy Squid filtre sit wèb pou elèv lekòl.
QoS

Quality of Service — priyorite trafik rezo (egzanp: voix > email).

Egzanp: QoS priyorite VoIP pou evite ko-koupaj pandan apèl.
Rack

Estrikti metalik pou òganize aparèy rezo (sèvè, switch, patch panel).

Egzanp: Yon rack 42U ka kenbe 42 aparèy 1U.
RIP

Routing Information Protocol — pwotokòl routing vye jenerasyon (maks 15 hops).

Egzanp: RIP pa touche pou gwo rezo — OSPF pi bon.
RJ45

Konnektè estanda pou kab Ethernet (8 broch).

Egzanp: Kab Cat6 avèk konnektè RJ45 konekte PC a switch.
Router

Aparèy Kouch 3 ki rote pakè ant rezo diferan.

Egzanp: Router lakay ou konekte LAN ou ak Entènèt ISP.
Routing Table

Tab nan routeur ki montre kote voye pakè selon destinasyon.

Egzanp: ip route show (Linux) oswa route print (Windows) montre routing table.
SFTP

Secure File Transfer Protocol — FTP chifre ak SSH. Port 22.

Egzanp: Administratè itilize SFTP pou transfere fichye konfidansyèl.
SMTP

Simple Mail Transfer Protocol — pou voye imel. Port 25.

Egzanp: Gmail itilize SMTP pou voye imel ou.
SNMP

Simple Network Management Protocol — monitore aparèy rezo. Port 161.

Egzanp: Sèvè Zabbix itilize SNMP pou kouri routeur yo.
SSH

Secure Shell — koneksyon adistans sekirize, chifre. Port 22.

Egzanp: ssh admin@192.168.1.1 = konekte ak routeur la sekirizman.
SSID

Service Set Identifier — non yon rezo Wi-Fi.

Egzanp: SSID 'MonRezo_5G' se non rezo Wi-Fi mwen an.
Static IP

Adrès IP manyèlman konfigire ki pa chanje.

Egzanp: Sèvè toujou gen IP estatik pou yo ka toujou jwenn yo.
Static Routing

Wout rezo konfigire manyèlman pa administratè.

Egzanp: ip route 10.0.0.0 255.0.0.0 192.168.1.254 = static route.
Subnet

Yon pòsyon rezo ki separe pa masque sou-rezo.

Egzanp: 192.168.1.0/24 ak 192.168.2.0/24 se de sou-rezo diferan.
Subnet Mask

Nimewo 32 bit ki separe pati rezo ak pati hôte nan yon adrès IP.

Egzanp: 255.255.255.0 (/24) = 24 bit pou rezo, 8 bit pou hôte.
Switch

Aparèy Kouch 2 ki konekte aparèy nan menm rezo, voye done espesifikman.

Egzanp: Switch 24 pò konekte tout PC nan yon biwo.
TCP

Transmission Control Protocol — pwotokòl fyab ak koneksyon (3-way handshake).

Egzanp: HTTP itilize TCP pou asire paj wèb rive konplètman.
Telnet

Pwotokòl aksè adistans NON chifre. Port 23. Pa sekirize.

Egzanp: Remplace Telnet ak SSH pou sekirite.
TFTP

Trivial File Transfer Protocol — FTP senp, pa chifre. Port 69.

Egzanp: Routeur Cisco itilize TFTP pou chaje firmware.
Topology

Aranjman fizik oswa lojik aparèy ak koneksyon nan yon rezo.

Egzanp: Star topology pi randan pou rezo kay.
Traceroute

Kòmand ki montre chak sote (hop) yon pakè fè pou rive destinasyon.

Egzanp: tracert google.com montre wout pakè la pran pou rive Google.
Trunk Port

Pò switch ki ka pote trafik plizyè VLAN an menm tan.

Egzanp: Lyen ant switch ak routeur se souvantou trunk port.
TTL

Time To Live — nimewo ki diminye nan chak routeur; pakè jete si TTL=0.

Egzanp: TTL pa default 128 (Windows) oswa 64 (Linux).
UDP

User Datagram Protocol — pwotokòl rapid san verifye livrezon.

Egzanp: DNS, DHCP, ak streaming vidyo itilize UDP.
UPS

Uninterruptible Power Supply — bèf kouran pou pwoteje aparèy kont koupe kouran.

Egzanp: UPS ba yon sèvè 10-30 minit backup kouran.
URL

Uniform Resource Locator — adrès konplè yon resous sou entènèt.

Egzanp: https://www.example.com/page se yon URL.
VLAN

Virtual LAN — segmentasyon lojik yon rezo fizik an plizyè rezo vityèl.

Egzanp: VLAN IT ak VLAN RH separe trafik ant depatman yo.
VPN

Virtual Private Network — tunel chifre sou yon rezo piblik.

Egzanp: Travayè adistans itilize VPN pou konekte sekirizman ak biwo a.
WAN

Wide Area Network — rezo ki kouvri distans long (ant vil, peyi).

Egzanp: Entènèt se pi gwo WAN ki egziste.
WPA2

Wi-Fi Protected Access 2 — sekirite Wi-Fi ak AES encryption.

Egzanp: Aktive WPA2-AES sou routeur ou pou sekirite Wi-Fi.
WPA3

Vèsyon pli nouvo WPA — pi solid pase WPA2.

Egzanp: WPA3 rezoud plizyè feblès WPA2 tankou KRACK.
Wireshark

Lojisyèl capture ak analize trafik rezo.

Egzanp: Administratè itilize Wireshark pou depane pwoblèm rezo konplèks.
802.11

Estanda IEEE pou Wi-Fi.

Egzanp: 802.11ac (Wi-Fi 5) ak 802.11ax (Wi-Fi 6) se estanda modèn.
802.1Q

Estanda IEEE pou VLAN tagging sou trunk ports.

Egzanp: 802.1Q ajoute yon 4-byte tag nan tèt frame pou idantifye VLAN.
802.3

Estanda IEEE pou Ethernet.

Egzanp: Ethernet 802.3 defini kòman done transmèt sou kab Ethernet.
3-Way Handshake

Pwosesis etabli koneksyon TCP: SYN, SYN-ACK, ACK.

Egzanp: Navigatè voye SYN, Sèvè reponn SYN-ACK, Navigatè fini ak ACK.

Okenn rezilta jwenn. Eseye yon lòt mo.
