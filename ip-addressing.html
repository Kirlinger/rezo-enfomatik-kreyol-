🔢 Adrès IP

Konprann kijan adrès IP travay — IPv4, IPv6, IP piblik vs prive, masque sou-rezo, CIDR, ak plis.

Kisa yon Adrès IP Ye?

Yon adrès IP (Internet Protocol address) se yon nimewo inik ki asiye bay chak aparèy nan yon rezo pou idantifye li. Se tankou yon adrès postal pou òdinatè ou — li pèmèt done jwenn destinasyon yo.

📬 Analoji Senp

Imajine Entènèt la se yon gwo vil ak milyon kay. Chak kay bezwen yon adrès pou faktè (lapòs) ka delivre pako yo. Adrès IP se adrès postal aparèy ou sou Entènèt la.

IPv4 – Version 4

IPv4 (Internet Protocol Version 4) se vèsyon ki pi souvan itilize. Li fòme de 4 okté (32 bits) separe pa yon pwen.

192.168.1.100

│ │ └──── Sou-rezo
│ └──────── Rezo Prive

⚠️
APIPA – 169.254.x.x:

Lè ou wè yon adrès 169.254.x.x, sa vle di aparèy la pa ka jwenn yon sèvè DHCP. Verifye koneksyon rezo ou ak sèvè DHCP a.

IP Estatik vs IP Dinamik

📌

IP Estatik

Adrès manyèlman konfigire ki pa chanje. Itilize pou sèvè, imprimant rezo, kamera sekirite.

  • Pi fasil pou jwenn toulejou
  • Mwens dependan sou DHCP
  • Pi chè si IP Piblik
🔄

IP Dinamik

Adrès otomatikman asiye pa DHCP. Chanje chak fwa aparèy la konekte. Pi souvan pou kliyan.

  • Pa bezwen konfigirasyon manyèl
  • Jere otomatikman
  • Adrès ka chanje

IPv6 – Version 6

IPv6 se vèsyon ki ranplase IPv4 paske adrès IPv4 yo fini. Li gen 128 bits konpare ak 32 bits pou IPv4. Li ekri an hexadecimal, separe pa pwen-divi.

# Egzanp adrès IPv6
2001:0db8:85a3:0000:0000:8a2e:0370:7334
# Fòm abrejye (zeros konsekitif = ::)
2001:db8:85a3::8a2e:370:7334
KarakteristikIPv4IPv6
Longè Adrès32 bits128 bits
Kantite Adrès~4 milya~340 undecilyon
FòmatDesimal puntuéHexadecimal pwen-divi
KonfigirasyonManyèl / DHCPSLAAC / DHCPv6
SekiriteOpsyonèl (IPSec)Entegre (IPSec)
NATNesesèPa nesesè

Tip Adrès IPv6 Kle yo

  • 2000::/3 – Global Unicast – IP piblik IPv6 (ekivalan IP piblik IPv4)
  • FE80::/10 – Link-Local – Otomatik sou chak entèfas (ekivalan 169.254.x.x)
  • ::1 – Loopback – Ekivalan 127.0.0.1
  • FF00::/8 – Multicast – Voye bay gwoup aparèy

Adrès MAC

Yon adrès MAC (Media Access Control) se yon identifyan inik 48-bit asiye nan fabrik nan chak NIC. Li opere nan Kouch 2 (Data Link).

# Fòmat Adrès MAC
00:1A:2B:3C:4D:5E
│──────────│ │──────────│
OUI (Fabrikant) Nimewo Idantifyan

# Kòman pou wè MAC address ou sou Windows
ipconfig /all
# Sou Linux/Mac
ip addr oswa ifconfig
💡
MAC vs IP:

Adrès MAC se pou komunikasyon lokal (Kouch 2). Adrès IP se pou komunikasyon rezo ak routage (Kouch 3). Les de travay ansanm.

Pasrèl Defo (Default Gateway)

Yon default gateway se adrès IP routeur lokal la. Lè yon aparèy vle kominike ak yon aparèy deyò rezo lokal li (pa egzanp, sou Entènèt), li voye pakè a bay gateway la.
