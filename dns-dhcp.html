🗂️ DNS ak DHCP

Konprann kijan DNS tradui non domèn an IP, ak kijan DHCP asiye adrès IP otomatik.

DNS – Domain Name System

DNS (Domain Name System) se "anyè telefòn" Entènèt la. Li tradui non domèn lisib pou moun (tankou google.com) an adrès IP nimerik (tankou 142.250.80.46) ke òdinatè yo ka itilize.

📞 Analoji: Anyè Telefòn

Imajine ou vle rele "Legliz Sentkatren". Ou pa konnen nimewo a — ou chèche nan anyè. DNS se menm bagay — ou tape facebook.com, DNS jwenn IP 157.240.x.x pou ou.

Kijan Rezolisyon DNS Fonksyone (Etap pa Etap)

1

Demann Kliyan

Ou tape www.google.com nan navigatè ou. Òdinatè ou mande sèvè DNS ou (jeneralman routeur ou oswa 8.8.8.8).

2

Cache Lokal

Èske repons lan nan cache lokal? Si wi, retounen IP dirèkteman. Si non, kontinye.

3

DNS Recursive Resolver

Sèvè DNS ISP ou mande yon Root Name Server kijan pou jwenn .com.

4

TLD Name Server

Root Server voye sèvè DNS la bay TLD Server pou .com ki di li kote jwenn google.com.

5

Authoritative Name Server

Sèvè otorite Google bay adrès IP reyèl pou www.google.com.

6

Repons bay Kliyan

Sèvè DNS la retounen IP la bay navigatè ou, ki konekte ak sèvè Google an dirèkteman.

DHCP – Dynamic Host Configuration Protocol

DHCP (Dynamic Host Configuration Protocol) otomatize asiyasyon adrès IP bay aparèy sou yon rezo. San DHCP, ou ta dwe konfigire manyèlman chak aparèy.

🏨 Analoji: Otèl

Lè ou rive nan yon otèl, resepsyon ba ou yon chanm (nimewo). Lè ou pati, chanm nan disponib pou lòt moun. Se menm bagay ak DHCP — li bay adrès IP tanporèrman, epi reprann li lè aparèy la pa la ankò.

Konfigirasyon DHCP tipik

# Konfigirasyon DHCP Scope
Scope: 192.168.1.0/24
Plag Asiyasyon: 192.168.1.100 – 192.168.1.200
Adrès Ekskli: 192.168.1.1 – 192.168.1.99 (estatik)
Pasrèl Defo: 192.168.1.1
Sèvè DNS: 8.8.8.8, 8.8.4.4
Dire Bèy (Lease): 8 jè (pa defo)

Nòt Teknik:

Mesaj Discover ak Offer yo voye sou pò UDP 67 (sèvè) ak pò 68 (kliyan). DHCP itilize UDP paske koneksyon pa etabli pou komansman — kliyan a pa gen IP ankò!

Bèy DHCP (DHCP Lease)

Yon bèy (lease) se dire pou yon IP asiye bay yon aparèy. Lè bèy la ekspire, kliyan la dwe renouvle li oswa mande yon nouvo IP. Si aparèy la pa renouvle, IP a disponib pou lòt.

DHCP Reservation

Ou ka konfigire yon sèvè DHCP pou toujou bay menm IP bay menm aparèy selon adrès MAC li. Sa bay avantaj IP prive san konfigirasyon manyèl sou chak aparèy.

# Reservation pou Imprimant rezo
Aparèy: Imprimant HP
MAC: 00:1A:2B:3C:4D:5E
IP Rezève: 192.168.1.50
# Chak fwa aparèy sa konekte, li resevwa 192.168.1.50
