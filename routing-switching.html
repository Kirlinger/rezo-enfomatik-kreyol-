🔄 Routing ak Switching

Konprann kijan routeur dirije pakè ak kijan switch jere trafik lokal, VLAN, ak inter-VLAN routing.

Diferans: Router vs Switch

🌐

Router

Konekte diferan rezo ak dirije (route) pakè ant yo. Opere nan Kouch 3 (Network). Itilize adrès IP.

  • Konekte LAN ak WAN
  • Route pakè sou baz IP
  • Jere tablo routing
  • Pwoteje rezo (NAT, Firewall debaz)
🔌

Switch

Konekte aparèy nan menm rezo lokal. Opere nan Kouch 2 (Data Link). Itilize adrès MAC.

  • Konekte PC, sèvè, imprimant
  • Dirije tram sou baz MAC
  • Jere tablo MAC (CAM table)
  • Sipòte VLAN

Kijan Routing Fonksyone

Lè yon routeur resevwa yon pakè, li:
1. Li adrès IP destinasyon an
2. Chèche nan tablo routing li
3. Voye pakè a nan bon entèfas (interface)

Tablo Routing (Routing Table)

# Kòmand pou wè tablo routing (Windows)
route print

# Kòmand pou wè tablo routing (Linux)
ip route show

# Egzanp Tablo Routing
Destination Gateway Masque Interface
0.0.0.0 192.168.1.1 0.0.0.0 eth0 # Default route
192.168.1.0 0.0.0.0 255.255.255.0 eth0 # Rezo lokal
10.0.0.0 192.168.1.254 255.255.0.0 eth0 # Rezo distans

Default Route (Wout Defo)

Yon default route (0.0.0.0/0) se wout pou voye nenpòt pakè ki pa gen yon wout espesifik. Rele tou "gateway of last resort". Se sa ki pèmèt ou reyini Entènèt.

Routing Estatik vs Routing Dinamik

📌

Routing Estatik

Administratè konfigire manyèlman chak wout. Pa chanje otomatikman si yon lyen tonbe.

# Ajoute wout estatik (Linux)
ip route add 10.0.0.0/8 via 192.168.1.1

Avantaj: Senp, previzib, mwens chaj CPU
Dezavantaj: Pa adapte otomatikman

🔄

Routing Dinamik

Routeur yo pataje enfòmasyon wout otomatikman ak chanje lè topoloji rezo chanje.

Pwotokòl:

VLAN – Virtual Local Area Network

Yon VLAN (Virtual LAN) pèmèt ou divize yon sèl switch fizik an plizyè rezo lokal lojik. Aparèy nan diferan VLAN pa ka kominike dirèkteman san yon routeur.

🏢 Analoji: Biwo

Imajine yon biwo ki gen depatman IT, Finansye, ak RH. Avèk VLAN, yo ka tout konekte nan menm switch men separe lojikman. IT pa ka wè trafik Finansye, ak Finansye pa ka wè trafik IT — san bezwen switch diferan.

Pò Switch: Access vs Trunk

🔑

Access Port

Yon pò ki manm yon sèl VLAN sèlman. Konekte pou aparèy final (PC, imprimant). Aparèy la pa konnen li nan yon VLAN.

# Konfigirasyon Cisco IOS
interface fa0/1
 switchport mode access
 switchport access vlan 10
🚗

Trunk Port

Yon pò ki pote plizyè VLAN an menm tan. Itilize ant switch ak switch, oswa switch ak routeur. Itilize IEEE 802.1Q pou etikete.

# Trunk Cisco IOS
interface fa0/24
 switchport mode trunk
 switchport trunk allowed vlan 10,20,30

Native VLAN

Yon Native VLAN se VLAN ki itilize pou trafik ki pa etikte sou yon trunk port. Pa defo, se VLAN 1. Pi bon pratik: chanje li pou yon VLAN inutilize pou sekirite.

⚠️
Sekirite VLAN:

Evite mete PC nan VLAN 1 (Native VLAN defo). Itilize VLAN separe ak bien konfigire pò trunk yo pou evite VLAN hopping attacks.

Inter-VLAN Routing

VLAN separe rezo yo — pou VLAN diferan kominike, ou bezwen yon routeur oswa yon Layer 3 Switch.

🔀

Router-on-a-Stick

Yon routeur ak yon sèl pò fizik ki konfigire an plizyè sous-entèfas lojik (sub-interfaces) — youn pou chak VLAN.

interface fa0/0.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0
🏗️

Layer 3 Switch

Yon switch ki ka fè routing tou. Pi efikas pou gwo rezo — routing se nan materyèl (hardware).

interface vlan 10
 ip address 192.168.10.1 255.255.255.0
ip routing
